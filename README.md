# 🤖 All-in-One Agent Bot

**Telegram এ সবকিছু একই বটে!**

## ✅ ফিচারস

| ফিচার | বিবরণ |
|--------|--------|
| 🎬 ভিডিও ডাউনলোড | YouTube, Facebook, Instagram, Twitter/X, TikTok, Vimeo — যেকোনো লিংক |
| 📊 Google Sheet | শিট নাম → কলাম → ডেটা → CSV ফাইল |
| 🔍 ভিডিও/গান সার্চ | YouTube সার্চ |
| 🌐 অনুবাদ | বাংলা ↔ ইংরেজি |
| 🔊 Text to Voice | Male/Female ভয়েস (বাংলা + ইংরেজি) |
| 🔗 URL Shortener | TinyURL দিয়ে ছোট করা |
| 📄 ফাইল তৈরি | যেকোনো নামে, যেকোনো কোড |

## 🚀 Deploy to Render

1. GitHub এ ৩টা ফাইল আপলোড করো
2. Render এ New Web Service → Connect GitHub
3. Environment: Python 3
4. Build Command: `pip install -r requirements.txt`
5. Start Command: `python bot.py`

**⚠️ Important:** Render এ Free Tier এ Sleep হয়ে যায়। 
Sleep এড়াতে UptimeRobot দিয়ে Ping রাখো।

## 📝 Bot Commands

- `/start` — মূল মেনু
- `/cancel` — বর্তমান কাজ বাতিল

## 🔑 Zero API Key Needed!

সব ফিচার ফ্রি — কোনো Paid API লাগে না!
